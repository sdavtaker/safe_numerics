Safe Numerics

Pending Issues

The library is under development. There are a number of issues still pending.

  • The library is currently limited to integers.

  • Conversions to safe integer types from floating point types is not checked.

  • Note that standard library stream conversion functions such as strtoi etc. DO check for valid input and throw the exception std::out_of_range if the string cannot be converted to the specified integer type. In other words, strtoi aready contains the exact functionality that safe<int> provides.
